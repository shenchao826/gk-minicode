# 国考资料工具 - 视频内容方案调研

## 小红书/抖音上国考类爆款视频的类型

### 类型一：图文轮播解说型（最推荐 ✅）
- **形式**：静态图片轮播 + AI配音 + 字幕
- **时长**：60-90秒
- **优势**：制作简单、信息密度高、完播率高
- **模板**：
  1. 封面（标题大字）→ 2. 痛点场景 → 3. 数据对比 → 4. 方法论 → 5. 工具引导
- **适合平台**：小红书（竖版9:16）、抖音
- **我们的配图**：已有 6 张 xhs 配图，可直接使用

### 类型二：动画录屏型
- **形式**：操作录屏 + 鼠标移动 + 画外音解说
- **时长**：2-5分钟
- **优势**：真实感强、转化率最高
- **缺点**：需要录屏软件，制作成本高
- **不适合当前阶段**

### 类型三：真人出镜讲解
- **形式**：真人面对镜头讲解
- **时长**：3-8分钟
- **优势**：信任感最强
- **缺点**：需要拍摄，不适合冷启动

### 类型四：AI 文生视频（技术展示型）
- **形式**：Agnes AI 生成的视频片段拼接
- **时长**：15-30秒
- **优势**：视觉冲击力强
- **缺点**：无声音、信息量少、不适合知识类内容
- **适合**：作为封面视频或引流素材

## 🏆 推荐方案：类型一（图文轮播解说型）

这是最适合我们现状的方案——**用已有的6张配图做轮播 + AI配音**。

### 具体做法

```
素材：6张 xhs 配图（xhs_bigtext.png ~ xhs_warning_circles.png）
配音：edge-tts 生成（免费、中文女声，类似真人）
字幕：逐行翻译内容（可选）
帧率：每张图片停留 3 秒
分辨率：1024x1792（9:16 竖版）
总时长：约 18 秒（纯轮播）→ 加长到 60 秒可放慢节奏
```

### 为什么不用"图生视频"？

| 对比项 | 文生图→图生视频 | 图文轮播解说 |
|--------|----------------|-------------|
| 信息传达 | ❌ 只有画面，无文字无解说 | ✅ 有字幕有配音 |
| 完播率 | ❌ 15秒没人看 | ✅ 内容丰富能看完 |
| 转化率 | ❌ 不知道你要卖什么 | ✅ 自然植入工具链接 |
| 制作成本 | ⚠️ 需要 ffmpeg 合并 | ✅ 简单拼接 |
| 适用平台 | 抖音/视频号 | 小红书/抖音/视频号 |

**结论：图文轮播解说更适合知识类产品推广。**

## 执行步骤

1. 用 edge-tts 生成 60 秒的解说音频
2. 6 张配图按顺序生成关键帧
3. 用 ffmpeg 拼接：图片轮播 + 配音 + 字幕
4. 输出 9:16 竖版视频
