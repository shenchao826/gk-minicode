# 岗策工具箱 - 资料生产工作流

## 一、整体生产流程

```
步骤1：大纲搭建 → 步骤2：AI批量出初稿 → 步骤3：人工二次修改去重脱敏 → 步骤4：排版封装 → 步骤5：打包分类
```

### 步骤1：大纲搭建
- 以国考考试大纲为唯一基准
- 划分目录结构，确定每个文档要包含的内容
- 明确每个文件的命名规范（禁用敏感词）

### 步骤2：AI批量出初稿
- 使用统一提示词模板（见 prompts/ 目录）
- 拒绝复制第三方网盘资料
- 确保内容原创重组改写

### 步骤3：人工二次修改去重脱敏
- 删除 AI 重复语句
- 替换所有和网红公考资料雷同的话术
- 文件内不能出现任何机构 logo、水印、讲师名字

### 步骤4：排版封装
- WPS 统一字体、行距、页眉
- 可编辑版：Word / Excel
- 不可转发版：PDF（添加浅色自定义水印：岗策工具箱整理）

### 步骤5：打包分类
- 按照网盘五级目录归类压缩
- 单个商品单独压缩包
- 生成网盘分享链接 + 提取码，录入后台卡密

## 二、五大板块生产计划

### 板块1：岗位报考筛选专区（引流爆款）
**目标产出：**
- 国考历年招录职位表（清洗排版版）
- 智能筛选Excel模板
- 报考流程指南包

**生产优先级：** 最高（半天可完成，快速上架）

**合规来源：** 国家公务员局官网招考简章

---

### 板块2：行测考点速记&专项工具（主力单品）
**目标产出：**
- 政治理论模块核心考点
- 言语理解与表达速记手册
- 数量关系公式速记表
- 判断推理解题思路
- 资料分析必背公式

**生产优先级：** 高（AI批量生成主力板块）

**合规来源：** 国考大纲 + AI原创梳理

---

### 板块3：申论写作素材&答题模板（利润核心）
**目标产出：**
- 归纳概括题规范词库
- 提出对策题模板
- 综合分析题框架
- 应用文公文格式模板
- 大作文素材库

**生产优先级：** 高（最容易量产，按月更新）

**合规来源：** 政府官网公文 + 人民日报评论摘抄

---

### 板块4：时政汇总&常识背诵手册（会员刚需）
**目标产出：**
- 年度时政大盘点
- 常识判断高频考点
- 碎片化背诵口袋文档

**生产优先级：** 中（长期可持续更新）

**合规来源：** 政府工作报告 + 新华社时政 + 初高中课本

---

### 板块5：面试+政审全套流程文档（延伸品类）
**目标产出：**
- 结构化面试答题思路
- 政审材料清单
- 体检须知手册

**生产优先级：** 中（纯信息整合，零成本）

**合规来源：** 国家公务员局公开公告
