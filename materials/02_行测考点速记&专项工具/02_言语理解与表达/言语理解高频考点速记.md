# 言语理解与表达高频考点速记

## 一、高频易混淆成语辨析

### （一）易混淆成语对比表

| 成语 | 释义 | 使用语境区分 |
|------|------|--------------|
| **不以为然** | 不认为是对的，表示不同意 | 侧重主观判断，不认可观点 |
| **不以为意** | 不把它放在心上，表示不重视 | 侧重态度，不在乎 |
| **差强人意** | 大体上还能让人满意 | 有一定满意，但不够好 |
| **尽如人意** | 完全符合人的心意 | 非常满意 |
| **刮目相看** | 用新的眼光来看待 | 强调变化后的新认识 |
| **另眼相看** | 用另一种眼光看待，多指看待某个人不同于一般 | 强调态度特殊 |
| **望尘莫及** | 只望见走在前面的人带起的尘土而追赶不上，比喻远远落后 | 侧重差距大，赶不上 |
| **望其项背** | 能够望见别人的颈项和后背，表示赶得上或比得上（多用于否定式） | 常用于"难以望其项背" |
| **不负众望** | 不辜负众人的期望 | 褒义，达到期望 |
| **不孚众望** | 不能使众人信服，未达到期望 | 贬义，未达到期望 |
| **一文不名** | 一个钱也没有，形容非常贫困 | 侧重贫穷 |
| **一文不值** | 指毫无价值 | 侧重价值低 |
| **惨淡经营** | 形容苦费心力地经营筹划 | 褒义，用心经营 |
| **经营惨淡** | 形容生意不好，不景气 | 贬义，经营状况差 |
| **不刊之论** | 指正确的、不可修改的言论 | 褒义，言论正确 |
| **不经之谈** | 荒唐无根据的话 | 贬义，言论荒谬 |
| **首当其冲** | 比喻最先受到攻击或灾难 | 中性，最先受影响 |
| **一马当先** | 形容领先或带头 | 褒义，积极带头 |
| **无可厚非** | 不可过分指责，表示虽有缺点但可以理解或原谅 | 侧重可以原谅 |
| **无可非议** | 没有什么可以指责的，表示言行合乎情理 | 侧重完全正确 |

### （二）高频实词辨析

| 实词 | 侧重点 | 搭配对象 |
|------|--------|----------|
| **颁布** | 公布（法令、条例等） | 法律、法令、条例 |
| **公布** | 公开发布，使大家知道 | 成绩、名单、结果 |
| **保障** | 保护（生命、财产、权利等），使不受侵犯 | 权利、安全、生活 |
| **保证** | 担保，确保做到 | 质量、完成、实现 |
| **爆发** | 突然发作，强调猛烈性 | 战争、革命、火山 |
| **暴发** | 突然发财或得势，强调突然性 | 山洪、疫情、户 |
| **制定** | 定出（法律、章程、计划等） | 计划、方案、法规 |
| **制订** | 创制拟定（尚未定案） | 草案、初稿 |
| **施行** | 法令、规章等公布后从某时起发生效力 | 法律、政策 |
| **实行** | 用行动来实现（纲领、政策、计划等） | 计划、制度、改革 |
| **遏制** | 阻止，控制（某事发展） | 势头、趋势 |
| **遏止** | 用力阻止（使停止） | 战争、洪流、行为 |

## 二、片段阅读解题思路

### （一）主旨概括题

**识别标志**：主旨、主要、核心、中心、概括、意在说明

**解题技巧**：

1. **找主题句**：
   - 关联词法：转折后、因果后、递进后是重点
   - 行文脉络法：总分、分总、总分总、分总分
   - 对策句通常是主旨

2. **排除干扰项**：
   - 排除过于绝对的选项
   - 排除片面的选项
   - 排除无关的选项

### （二）意图判断题

**识别标志**：意在、想要、推出、启示

**解题技巧**：

1. 主旨是问题→意图是解决问题的对策
2. 主旨是观点→意图是观点本身
3. 主旨是故事→意图是寓意启示

### （三）细节判断题

**识别标志**：正确/不正确、符合/不符合、错误的是

**常见陷阱**：

1. 偷换概念：替换词语、扩大/缩小范围
2. 偷换时态：过去/现在/将来混淆
3. 偷换数量：全部/部分混淆
4. 偷换语气：肯定/否定混淆
5. 无中生有：原文没有的内容

## 三、语句表达做题步骤

### （一）语句排序题

**做题步骤**：

1. **确定首句**：
   - 适合做首句：背景介绍、概念定义、观点引入
   - 不适合做首句：指代不明的代词、补充说明的词（也、又、再）

2. **寻找捆绑集团**：
   - 关联词捆绑：因果、转折、递进
   - 指代词捆绑：这、那、其、他们
   - 话题一致捆绑：同一话题的句子放在一起

3. **确定尾句**：
   - 结论句、对策句适合做尾句
   - 补充说明的句子不适合做尾句

### （二）语句填空题

**做题步骤**：

1. **看位置**：
   - 段首：总起全文，引出下文
   - 段中：承上启下，衔接上下文
   - 段尾：总结全文，点明主旨

2. **看话题**：保持与前后文话题一致

3. **看逻辑**：遵循因果、转折、递进等逻辑关系

## 四、做题技巧总结

### （一）快速阅读技巧

1. **略读**：快速浏览首尾句、关联词、标点符号
2. **跳读**：跳过例子、数据、引用等次要内容
3. **精读**：重点阅读主题句、对策句、观点句

### （二）选项排除技巧

1. **排除绝对化表述**：所有、全部、一定、必须
2. **排除无中生有**：原文未提及的内容
3. **排除偷换概念**：选项与原文概念不一致
4. **排除片面选项**：只涵盖部分内容

### （三）时间分配建议

| 题型 | 每题建议时间 |
|------|-------------|
| 逻辑填空 | 30-40秒 |
| 片段阅读 | 40-50秒 |
| 语句表达 | 40-50秒 |
