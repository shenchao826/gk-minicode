# 资料分析必背公式速记

## 一、基础统计术语

### （一）核心概念

- **现期**：当前时期的数据
- **基期**：作为比较基准的时期数据
- **增长量**：现期量 - 基期量
- **增长率**：(现期量 - 基期量)/基期量 × 100%

### （二）特殊增长率

- **同比增长率**：与上年同期相比的增长率
- **环比增长率**：与上一统计周期相比的增长率
- **年均增长率**：一段时间内平均每年的增长率

### （三）比重相关

- **比重**：部分量/总体量 × 100%
- **平均数**：总体量/个数
- **倍数**：A/B

## 二、必背公式大全

### （一）增长率相关

| 公式 | 说明 |
|------|------|
| **增长率(r)** = (现期量 - 基期量)/基期量 | 基本公式 |
| **增长率(r)** = 增长量/基期量 | 等价公式 |
| **增长率(r)** = (现期量/基期量) - 1 | 等价公式 |
| **现期量** = 基期量 × (1 + r) | 已知基期求现期 |
| **基期量** = 现期量/(1 + r) | 已知现期求基期 |
| **增长量** = 基期量 × r | 已知基期求增量 |
| **增长量** = 现期量 × r/(1 + r) | 已知现期求增量 |

### （二）间隔增长

| 公式 | 说明 |
|------|------|
| **间隔增长率** = r₁ + r₂ + r₁ × r₂ | 两年间隔增长率 |
| **间隔倍数** = 1 + r₁ + r₂ + r₁ × r₂ | 间隔倍数 |

### （三）混合增长

| 公式 | 说明 |
|------|------|
| **混合增长率**：介于两部分增长率之间，偏向基数大的一方 | 十字交叉法 |

### （四）比重相关

| 公式 | 说明 |
|------|------|
| **比重** = 部分量/总体量 × 100% | 基本公式 |
| **部分量** = 总体量 × 比重 | 已知总体求部分 |
| **总体量** = 部分量/比重 | 已知部分求总体 |
| **基期比重** = (部分现期量/总体现期量) × [(1+总体增长率)/(1+部分增长率)] | 基期比重 |
| **比重变化量** = (部分现期量/总体现期量) × [(部分增长率 - 总体增长率)/(1+部分增长率)] | 比重变化 |

### （五）平均数相关

| 公式 | 说明 |
|------|------|
| **平均数** = 总体量/个数 | 基本公式 |
| **基期平均数** = (总体现期量/个数现期量) × [(1+个数增长率)/(1+总体增长率)] | 基期平均数 |
| **平均数增长率** = (总体增长率 - 个数增长率)/(1+个数增长率) | 平均数增长率 |

### （六）倍数相关

| 公式 | 说明 |
|------|------|
| **现期倍数** = A/B | 基本公式 |
| **基期倍数** = (A/B) × [(1+B增长率)/(1+A增长率)] | 基期倍数 |

### 速算技巧

- **截位直除**：根据选项差距，保留1-2位有效数字
- **分数比较**：分子大分母小则分数大
- **估算**：接近的分数替换，如333≈1/3，143≈1/7

## 四、常见陷阱总结

### （一）时间陷阱

1. **同比vs环比**：混淆同比和环比数据
2. **时间范围**：题目问的是上半年还是全年
3. **累计数据**：图表可能是累计值而非当期值

### （二）单位陷阱

1. **量级差异**：亿元/万元/元混淆
2. **百分比vs百分点**：百分点是两个百分比的差值
3. **人口单位**：万人/千人/人

### （三）题干陷阱

1. **范围限定**："超过"、"不到"、"至少"
2. **主体混淆**：甲的增长率vs乙的增长率
3. **概念偷换**：比重vs增长率、平均数vs中位数

### （四）计算陷阱

1. **增长率vs倍数**：增长率+1=倍数
2. **比重变化vs比重值**：变化量是百分点，不是百分比
3. **平均数vs中位数**：概念不同

## 五、做题流程

### （一）读题三步法

1. **看时间**：确定是现期还是基期
2. **找主体**：定位数据对应的名词
3. **判题型**：确定用哪个公式

### （二）计算优先级

1. **观察选项**：确定精度要求
2. **估算优先**：能估算不精算
3. **排除法**：根据量级、正负排除

### （三）时间分配

| 题型 | 每题建议时间 |
|------|-------------|
| 直接查找 | 20-30秒 |
| 简单计算 | 30-40秒 |
| 增长率计算 | 40-50秒 |
| 比重计算 | 40-50秒 |
| 综合分析 | 60-80秒 |

## 六、公式速记表

### 核心公式一览表

| 类别 | 公式 | 适用场景 |
|------|------|----------|
| 增长率 | r = (现-基)/基 | 基本增长率计算 |
| 现期量 | 现 = 基×(1+r) | 已知基期求现期 |
| 基期量 | 基 = 现/(1+r) | 已知现期求基期 |
| 增长量 | 增 = 现×r/(1+r) | 已知现期求增量 |
| 间隔增长 | r₁₂ = r₁ + r₂ + r₁×r₂ | 两年间隔增长率 |
| 基期比重 | B×[(1+R)/(1+r)] | 求基期比重 |
| 比重变化 | B×[(r-R)/(1+r)] | 比重变化量 |
| 平均增长率 | (R-r)/(1+r) | 平均数增长率 |
