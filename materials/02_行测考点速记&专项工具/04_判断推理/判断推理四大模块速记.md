# 判断推理四大模块速记

## 一、图形推理

### （一）位置规律

#### 1. 平移

**核心要点**：方向、步数、路径

| 类型 | 说明 | 示例 |
|------|------|------|
| 方向 | 上下、左右、对角线、顺时针、逆时针 | 元素沿水平方向移动 |
| 步数 | 恒定步数、递增步数、递减步数 | 每次移动1格、2格、3格... |
| 路径 | 循环（到头返回）、反弹（到头反向） | 元素从左到右循环移动 |

**例题示范**：

```
例：从所给选项中，选择最合适的一个填入问号处，使之呈现一定规律性：

图1：●在左上角
图2：●在右上角
图3：●在右下角
图4：●在左下角
图5：？

答案：●回到左上角，呈顺时针移动规律
```

#### 2. 旋转

**核心要点**：方向、角度

| 类型 | 说明 | 示例 |
|------|------|------|
| 方向 | 顺时针、逆时针 | 图形顺时针旋转 |
| 角度 | 45°、90°、180°、270° | 每次旋转90° |

**例题示范**：

```
例：图形每次顺时针旋转90°，下一个图形应是原图形旋转90°后的样子
```

#### 3. 翻转

**核心要点**：水平翻转、垂直翻转

| 类型 | 说明 | 示例 |
|------|------|------|
| 水平翻转 | 左右翻转，像照镜子 | "b"水平翻转后像"d" |
| 垂直翻转 | 上下翻转 | "b"垂直翻转后像"p" |

**例题示范**：

```
例：图形先水平翻转，再垂直翻转，下一个图形应是原图形垂直翻转后的样子
```

**练习题**：

1. 元素每次向右移动2格，到头后回到左边继续，第5个图元素位置？
2. 图形每次逆时针旋转45°，第4个图的样子？
3. 图形先水平翻转，再垂直翻转，第3个图的样子？

### （二）样式规律

#### 1. 遍历

**核心要点**：元素重复出现，缺啥补啥

**例题示范**：

```
例：第一行：○△□
    第二行：□○△
    第三行：△□？
    
答案：○（每行都包含○△□，第三行缺少○）
```

#### 2. 加减同异

| 类型 | 说明 | 示例 |
|------|------|------|
| 相加 | 两个图形叠加 | 图1+图2=图3 |
| 相减 | 图1减去图2的部分 | 图1-图2=图3 |
| 求同 | 保留相同部分 | 保留图1和图2都有的部分 |
| 求异 | 保留不同部分 | 保留图1和图2不同的部分 |

**例题示范**：

```
例：图1和图2求异得到图3
图1：○△
图2：○□
图3：△□（保留不同的△和□）
```

#### 3. 黑白运算

**核心要点**：黑+黑、黑+白、白+黑、白+白

**例题示范**：

```
例：已知：黑+白=黑，白+黑=黑，黑+黑=白，白+白=白
图1：黑块+图2：白块=图3：黑块
```

**练习题**：

1. 第一组：图1+图2=图3，第二组按此规律，图3应为？
2. 第一组：图1和图2求同得到图3，第二组按此规律，图3应为？
3. 黑白运算：黑+黑=白，白+白=黑，黑+白=黑，求图3？

### （三）数量规律

#### 1. 点

**核心要点**：交点数、切点、端点

**例题示范**：

```
例：图形交点数依次为1、2、3、4、？，下一个图形应有5个交点
```

#### 2. 线

**核心要点**：直线数、曲线数、笔画数

**笔画数判断方法**：
- 奇点：从该点出发的线条数为奇数
- 笔画数=奇点数÷2（奇点为0或2时，为一笔画）

**例题示范**：

```
例：图形笔画数依次为1、2、1、2、？，下一个图形应为一笔画图形
```

#### 3. 角

**核心要点**：直角数、锐角数、钝角数

**例题示范**：

```
例：图形直角数依次为1、2、3、4、？，下一个图形应有5个直角
```

#### 4. 面

**核心要点**：封闭区域数、面积大小

**例题示范**：

```
例：图形面数依次为1、2、3、4、？，下一个图形应有5个面
```

#### 5. 素

**核心要点**：元素个数、种类数、部分数

**例题示范**：

```
例：图形元素种类数依次为1、2、3、4、？，下一个图形应有5种元素
```

**练习题**：

1. 图形交点数依次为2、3、4、5、？，下一个图形应有几个交点？
2. 判断图形是否为一笔画图形：☆（五角星）
3. 图形面数依次为3、4、5、6、？，下一个图形应有几个面？

### （四）属性规律

#### 1. 对称性

**核心要点**：轴对称、中心对称、轴+中心对称

| 类型 | 说明 | 示例 |
|------|------|------|
| 轴对称 | 沿对称轴折叠后重合 | ★、等腰三角形 |
| 中心对称 | 绕中心旋转180°后重合 | ◇、平行四边形 |
| 轴+中心对称 | 同时满足轴对称和中心对称 | ○、□ |

**例题示范**：

```
例：所有图形都是轴对称图形，且对称轴数量依次增加
```

#### 2. 曲直性

**核心要点**：全直、全曲、曲直混合

**例题示范**：

```
例：第一组全直，第二组全曲，第三组曲直混合
```

#### 3. 开闭性

**核心要点**：全封闭、全开放、半开半闭

**例题示范**：

```
例：图形依次为全封闭、全开放、全封闭、全开放、？，下一个应为全封闭
```

**练习题**：

1. 判断图形对称性：☆（五角星）、◇（菱形）、○（圆）
2. 图形曲直性依次为全直、全曲、全直、全曲、？，下一个应为？
3. 图形开闭性依次为全封闭、全开放、半开半闭、全封闭、？，下一个应为？

### （五）空间规律

#### 1. 六面体

**核心要点**：相对面不相邻、相邻面位置关系、公共点法

**相对面判断方法**：
- 同行或同列隔一个面
- "Z"字形两端的面

**相邻面判断方法**：
- 看公共边、公共点
- 看面内元素的相对位置

**例题示范**：

```
例：左边给定的是正方体的外表面展开图，右边哪一项能由它折叠而成？
```

#### 2. 三视图

**核心要点**：主视图、俯视图、左视图

**例题示范**：

```
例：左边立体图形，右边哪一项是它的主视图？
```

#### 3. 截面图

**核心要点**：常见立体图形截面

| 立体图形 | 可能的截面 | 不可能的截面 |
|----------|------------|--------------|
| 正方体 | 正方形、长方形、三角形、六边形 | 直角三角形 |
| 圆柱 | 圆形、长方形、椭圆形 | 三角形 |
| 圆锥 | 圆形、三角形、椭圆形 | 长方形 |

**练习题**：

1. 判断正方体展开图中哪两个面是相对面
2. 画出圆柱的主视图和俯视图
3. 判断正方体能否截出直角三角形

## 二、类比推理

### （一）语义关系

#### 1. 近义词

**例题示范**：

```
例：美丽∶漂亮
A. 快乐∶高兴
B. 悲伤∶愤怒
C. 聪明∶愚蠢
D. 勇敢∶懦弱

答案：A（美丽和漂亮是近义词，快乐和高兴也是近义词）
```

#### 2. 反义词

**例题示范**：

```
例：聪明∶愚蠢
A. 美丽∶漂亮
B. 快乐∶悲伤
C. 勇敢∶英勇
D. 善良∶友善

答案：B（聪明和愚蠢是反义词，快乐和悲伤也是反义词）
```

#### 3. 比喻象征

**例题示范**：

```
例：玫瑰∶爱情
A. 百合∶纯洁
B. 菊花∶坚强
C. 牡丹∶富贵
D. 梅花∶傲骨

答案：A（玫瑰象征爱情，百合象征纯洁）
```

**练习题**：

1. 认真∶仔细（选择近义词）

### （二）逻辑关系

#### 1. 全同关系

**例题示范**：

```
例：土豆∶马铃薯
A. 番茄∶西红柿
B. 苹果∶香蕉
C. 牛奶∶酸奶
D. 米饭∶面条

答案：A（土豆和马铃薯是同一事物的不同名称）
```

#### 2. 并列关系

**矛盾关系 vs 反对关系**：

| 类型 | 说明 | 示例 |
|------|------|------|
| 矛盾关系 | 非此即彼，没有第三种可能 | 生-死、男-女 |
| 反对关系 | 有第三种可能 | 红-蓝、苹果-香蕉 |

**例题示范**：

```
例1（矛盾关系）：生∶死
例2（反对关系）：红∶蓝
```

#### 3. 包容关系

**组成关系 vs 种属关系**：

| 类型 | 说明 | 示例 |
|------|------|------|
| 组成关系 | 整体与部分 | 电脑-键盘、汽车-轮胎 |
| 种属关系 | 大类与小类 | 水果-苹果、动物-猫 |

**例题示范**：

```
例1（组成关系）：电脑∶键盘
例2（种属关系）：水果∶苹果
```

#### 4. 交叉关系

**例题示范**：

```
例：党员∶教师
A. 男人∶女人
B. 医生∶护士
C. 青年∶学生
D. 工人∶农民

答案：C（有的党员是教师，有的教师是党员，交叉关系）
```

#### 5. 对应关系

| 类型 | 说明 | 示例 |
|------|------|------|
| 功能对应 | 事物的用途 | 剪刀-裁剪、汽车-运输 |
| 场所对应 | 事物所在的地点 | 医生-医院、学生-学校 |
| 时间顺序 | 事件发生的先后顺序 | 播种-收获、日出-日落 |
| 因果关系 | 原因和结果 | 下雨-地湿、努力-成功 |
| 原材料对应 | 制作事物的材料 | 面粉-面包、木材-桌子 |

**例题示范**：

```
例1（功能对应）：剪刀∶裁剪
例2（因果关系）：下雨∶地湿
例3（原材料对应）：面粉∶面包
```

**练习题**：

1. 北京∶中国首都（全同关系）
2. 老虎∶动物（种属关系）
3. 轮胎∶汽车（组成关系）
4. 作家∶青年（交叉关系）
5. 牙刷∶刷牙（功能对应）

## 三、定义判断

### （一）抓关键词法

**核心关键词**：

| 关键词类型 | 说明 | 示例 |
|------------|------|------|
| 主体 | 动作的发出者 | 政府、企业、个人、组织 |
| 客体 | 动作的承受者 | 消费者、学生、患者 |
| 条件 | 前提、假设、限定词 | 在...条件下、当...时 |
| 方式 | 通过、利用、按照 | 通过网络、利用技术 |
| 目的 | 为了、达到、实现 | 为了提高效率、达到目标 |
| 结果 | 导致、造成、产生 | 导致经济增长、造成环境污染 |

**例题示范**：

```
例：行政决策是指行政机关及其工作人员为了实现行政目标，依照法律规定的权限和程序，对涉及公共利益的事项作出决定的行为。

关键词：
- 主体：行政机关及其工作人员
- 目的：实现行政目标
- 方式：依照法律规定的权限和程序
- 对象：涉及公共利益的事项
- 结果：作出决定

选项判断：
A. 某企业制定发展规划（主体不符，排除）
B. 某行政机关对公共交通票价作出调整（符合所有关键词，正确）
C. 某个人决定购买商品（主体不符，排除）
```

### （二）做题步骤

1. **看提问**：明确选是题还是选非题
2. **读题干**：划出核心关键词
3. **对比选项**：逐一匹配关键词

**例题示范**：

```
例：题目要求：选出属于"故意杀人"的选项

题干定义：故意杀人是指故意非法剥夺他人生命的行为。

关键词：故意、非法、剥夺他人生命

选项A：张三不小心导致他人死亡（不是故意，排除）
选项B：李四故意非法剥夺他人生命（符合所有关键词，正确）
选项C：王五依法执行死刑（不是非法，排除）
```

### （三）快速排除技巧

| 技巧 | 说明 | 示例 |
|------|------|------|
| 主体不符 | 选项中的主体与定义中的主体不一致 | 定义主体是政府，选项主体是企业 |
| 无关键词 | 选项中缺少定义的核心要素 | 定义要求"故意"，选项没有体现 |
| 矛盾表述 | 选项与定义中的某个关键词冲突 | 定义要求"非法"，选项是"合法" |

**练习题**：

1. 定义：教育是指教育者根据一定的社会要求，有目的、有计划、有组织地对受教育者的身心施加影响，使其朝着所期望的方向发展的活动。
   - 选项：父母教孩子说话（是否属于教育？）
2. 定义：商品是指用于交换的劳动产品。
   - 选项：农民种的自己吃的蔬菜（是否属于商品？）

## 四、逻辑判断

### （一）翻译推理

#### 1. 翻译规则

| 关联词 | 翻译方式 | 示例 |
|--------|----------|------|
| 如果A，那么B | A→B | 如果下雨，地就湿（下雨→地湿） |
| 只有A，才B | B→A | 只有努力，才能成功（成功→努力） |
| 只要A，就B | A→B | 只要坚持，就能胜利（坚持→胜利） |
| 除非A，否则B | ¬A→B | 除非学习，否则落后（¬学习→落后） |
| A是B的必要条件 | B→A | 努力是成功的必要条件（成功→努力） |
| A是B的充分条件 | A→B | 下雨是地湿的充分条件（下雨→地湿） |

#### 2. 推理规则

**核心推理规则**：

| 规则 | 说明 | 示例 |
|------|------|------|
| 逆否等价 | A→B ≡ ¬B→¬A | 如果下雨→地湿，那么地不湿→没下雨 |
| 传递规则 | A→B，B→C → A→C | 如果下雨→地湿，地湿→路滑，那么下雨→路滑 |
| 否定前件 | A→B，¬A → 不确定 | 如果下雨→地湿，没下雨，地可能湿也可能不湿 |
| 肯定后件 | A→B，B → 不确定 | 如果下雨→地湿，地湿了，可能下雨也可能没下雨 |

**例题示范**：

```
例：如果天下雨，地面就会湿。地面没有湿，所以（ ）

解：下雨→地湿，逆否等价：¬地湿→¬下雨
答案：天没有下雨
```

#### 3. 联言命题

**核心要点**：

- **A且B**：全真才真，一假即假
- **矛盾**：¬A或¬B

**例题示范**：

```
例：小张既会唱歌又会跳舞（唱歌且跳舞）
矛盾：小张不会唱歌或不会跳舞
```

#### 4. 选言命题

**核心要点**：

- **A或B**：一真即真，全假才假
- **矛盾**：¬A且¬B
- **否定肯定式**：A或B，¬A → B

**例题示范**：

```
例：小张或者会唱歌，或者会跳舞（唱歌或跳舞）
否定肯定式：如果小张不会唱歌，那么他会跳舞
```

**练习题**：

1. 如果努力学习，就能考上公务员。小王考上了公务员，所以小王努力学习了（是否正确？）
2. 只有身体健康，才能参加比赛。小李参加了比赛，所以小李身体健康（是否正确？）
3. 除非下雨，否则比赛正常进行。比赛没有正常进行，所以下雨了（是否正确？）
4. 小张会唱歌且会跳舞的矛盾命题是什么？
5. 小张会唱歌或会跳舞的否定肯定式是什么？

### （二）加强削弱

#### 1. 加强方式

| 方式 | 力度 | 说明 | 示例 |
|------|------|------|------|
| 搭桥 | 最强 | 建立论据与论点的联系 | 论据：小明成绩好，论点：小明能考上大学，搭桥：成绩好的人能考上大学 |
| 补充论据 | 中等 | 增加新证据支持论点 | 论点：运动有益健康，补充论据：研究表明运动能降低心脏病风险 |
| 必要条件 | 中等 | 没有它论点不成立 | 论点：要成功必须努力，必要条件：不努力一定不会成功 |
| 举例论证 | 较弱 | 通过例子支持 | 论点：努力能成功，举例：张三努力后成功了 |

**例题示范**：

```
例：论点：多吃水果有益健康
论据：水果富含维生素

加强方式：
1. 搭桥：维生素对健康有益
2. 补充论据：研究表明多吃水果的人免疫力更强
3. 必要条件：不吃水果会导致维生素缺乏
4. 举例论证：小明每天吃水果，身体很健康
```

#### 2. 削弱方式

| 方式 | 力度 | 说明 | 示例 |
|------|------|------|------|
| 拆桥 | 最强 | 切断论据与论点的联系 | 论据：小明成绩好，论点：小明能考上大学，拆桥：成绩好的人不一定能考上大学 |
| 否定论点 | 最强 | 直接反驳论点 | 论点：运动有益健康，否定：运动对健康没有好处 |
| 否定论据 | 中等 | 质疑论据真实性 | 论点：运动有益健康，论据：研究表明运动有益，否定论据：该研究不科学 |
| 因果倒置 | 较强 | 指出因果关系颠倒 | 论点：因为运动所以健康，因果倒置：因为健康所以运动 |
| 另有他因 | 中等 | 提出其他原因 | 论点：因为运动所以健康，另有他因：健康是因为饮食好 |

**例题示范**：

```
例：论点：喝牛奶能长高
论据：喝牛奶的孩子比不喝牛奶的孩子长得高

削弱方式：
1. 拆桥：喝牛奶和长高之间没有必然联系
2. 否定论点：喝牛奶不能让人长高
3. 否定论据：该研究样本不具有代表性
4. 因果倒置：不是因为喝牛奶所以长高，而是因为长高才喝牛奶
5. 另有他因：喝牛奶的孩子营养更好，所以长高
```

**练习题**：

1. 论点：读书能提高智力，论据：读书多的人智力测试得分高，如何加强？
2. 论点：吸烟导致肺癌，论据：吸烟的人肺癌发病率高，如何削弱？

### （三）真假推理

#### 1. 找矛盾关系

**常见矛盾关系**：

| 类型 | 矛盾关系 | 示例 |
|------|----------|------|
| A和¬A | A和非A | 小明是学生和小明不是学生 |
| A且B和¬A或¬B | A且B和非A或非B | 小明既聪明又努力和小明不聪明或不努力 |
| A或B和¬A且¬B | A或B和非A且非B | 小明聪明或努力和小明不聪明且不努力 |
| 所有A是B和有的A不是B | 所有A是B和有的A不是B | 所有学生都努力和有的学生不努力 |
| 有的A是B和所有A不是B | 有的A是B和所有A不是B | 有的学生努力和所有学生都不努力 |

**例题示范**：

```
例：甲说："我是学生"，乙说："甲不是学生"，已知只有一人说真话，则（ ）

解：甲和乙的话是矛盾关系（A和¬A），必有一真一假
因为只有一人说真话，所以甲说假话，乙说真话
答案：甲不是学生
```

#### 2. 找反对关系

**常见反对关系**：

| 类型 | 说明 | 示例 |
|------|------|------|
| 所有A是B和所有A不是B | 至少一假 | 所有学生都努力和所有学生都不努力 |
| 有的A是B和有的A不是B | 至少一真 | 有的学生努力和有的学生不努力 |

**例题示范**：

```
例：甲说："所有学生都努力"，乙说："所有学生都不努力"，已知只有一人说假话，则（ ）

解：甲和乙的话是反对关系（所有A是B和所有A不是B），至少一假
因为只有一人说假话，所以甲说真话，乙说假话
答案：所有学生都努力
```

#### 3. 假设验证

**例题示范**：

```
例：甲、乙、丙三人中只有一人会开车
甲说："我会开车"
乙说："我不会开车"
丙说："甲不会开车"
已知只有一人说真话，则（ ）

解：假设甲说真话（甲会开车），则乙和丙都说假话
乙说假话→乙会开车，与只有一人会开车矛盾
假设乙说真话（乙不会开车），则甲和丙都说假话
甲说假话→甲不会开车，丙说假话→甲会开车，矛盾
假设丙说真话（甲不会开车），则甲和乙都说假话
甲说假话→甲不会开车，乙说假话→乙会开车，符合条件
答案：乙会开车
```

**练习题**：

1. 甲说："今天下雨"，乙说："今天不下雨"，已知只有一人说真话，今天是否下雨？
2. 甲说："所有同学都及格了"，乙说："有的同学没及格"，已知只有一人说真话，同学是否都及格了？

### （四）分析推理

#### 1. 排除法

**例题示范**：

```
例：甲、乙、丙三人分别来自北京、上海、广州
甲不是北京人，乙不是上海人，丙是广州人
问三人分别来自哪里？

解：丙是广州人（确定）
甲不是北京人→甲是上海人
乙不是上海人→乙是北京人
答案：甲-上海，乙-北京，丙-广州
```

#### 2. 最大信息法

**例题示范**：

```
例：甲、乙、丙三人分别从事教师、医生、律师职业
甲和医生不同岁，医生比乙年轻，丙比律师年龄大
问三人分别从事什么职业？

解：医生出现次数最多（2次），从医生入手
甲和医生不同岁→甲不是医生
医生比乙年轻→乙不是医生→丙是医生
丙（医生）比乙年轻，丙比律师年龄大→乙不是律师→乙是教师
甲是律师
答案：甲-律师，乙-教师，丙-医生
```

#### 3. 列表法

**例题示范**：

```
例：甲、乙、丙、丁四人分别喜欢篮球、足球、羽毛球、乒乓球
甲不喜欢篮球，乙喜欢足球，丙不喜欢羽毛球，丁喜欢乒乓球
问四人分别喜欢什么运动？

解：列表整理：
    篮球 足球 羽毛球 乒乓球
甲  ×    ?    ?      ?
乙  ?    √    ?      ?
丙  ?    ?    ×      ?
丁  ?    ?    ?      √

乙喜欢足球，丁喜欢乒乓球
甲不喜欢篮球→甲喜欢羽毛球
丙喜欢篮球
答案：甲-羽毛球，乙-足球，丙-篮球，丁-乒乓球
```

#### 4. 假设法

**例题示范**：

```
例：甲、乙、丙三人中有一人是小偷
甲说："我不是小偷"
乙说："甲是小偷"
丙说："我不是小偷"
已知只有一人说假话，则小偷是谁？

解：假设甲是小偷，则甲说假话，乙说真话，丙说真话，符合条件
假设乙是小偷，则甲说真话，乙说假话，丙说真话，符合条件
假设丙是小偷，则甲说真话，乙说假话，丙说假话，不符合条件

需要更多条件才能确定，题目条件不足
```

**练习题**：

1. 甲、乙、丙三人分别是教师、医生、工程师，甲比医生年龄大，乙和工程师不同岁，工程师比丙年龄小，问三人职业？
2. 甲、乙、丙、丁四人分别住在1-4楼，甲不住1楼，乙住2楼，丙不住3楼，丁住4楼，问四人分别住几楼？

## 五、判断推理答题技巧

### （一）时间分配

- 图形推理：每题40-50秒
- 类比推理：每题30-40秒
- 定义判断：每题1分钟
- 逻辑判断：每题1-1.5分钟

### （二）答题策略

1. **先易后难**：先做类比推理和定义判断，再做图形推理和逻辑判断
2. **排除法优先**：通过排除明显错误选项提高正确率
3. **抓关键词**：定义判断和逻辑判断中，抓住核心关键词

### （三）常见陷阱

1. **翻译推理陷阱**：混淆充分条件和必要条件
2. **加强削弱陷阱**：偷换概念、无关选项
3. **真假推理陷阱**：矛盾关系和反对关系混淆
4. **分析推理陷阱**：信息遗漏、假设错误

## 六、必背逻辑公式

### （一）翻译推理

- **A→B** 的逆否命题：**¬B→¬A**
- **A且B** 的矛盾：**¬A或¬B**
- **A或B** 的矛盾：**¬A且¬B**
- **A或B** 的否定肯定式：**¬A→B**，**¬B→A**

### （二）模态命题

- **必然A** 的矛盾：**可能¬A**
- **可能A** 的矛盾：**必然¬A**
- **必然A** 的等价：**不可能¬A**
- **可能A** 的等价：**不必然¬A**

### （三）集合推理

- **所有A是B** → **有的B是A**
- **所有A不是B** → **所有B不是A**
- **有的A是B** → **有的B是A**
- **有的A不是B** 不能推出 **有的B不是A**

### （四）矛盾关系速记

| 原命题 | 矛盾命题 |
|--------|----------|
| A | ¬A |
| A且B | ¬A或¬B |
| A或B | ¬A且¬B |
| 所有A是B | 有的A不是B |
| 有的A是B | 所有A不是B |
| 必然A | 可能¬A |
| 可能A | 必然¬A |
