国考报考全流程指南

前言

本指南详细梳理国家公务员考试从公告发布到最终录用的完整流程，涵盖每个环节的具体操作步骤、时间节点、注意事项及所需材料清单，帮助考生顺利完成报考。

一、公告发布阶段

1.1 公告发布时间

  • 发布时间：通常在每年10月中下旬（以当年官方通知为准）
  • 发布平台：国家公务员局官网（www.scs.gov.cn）考录专题页面
  • 查询方式：登录官网 → 进入"中央机关及其直属机构XX年度考试录用公务员专题" → 查看公告

1.2 公告核心内容解读

（1）报考条件

（2）重要时间节点（以2026年为例）

（3）笔试内容

  • 公共科目：行政职业能力测验（行测）+ 申论
  • 专业科目：部分岗位需加试（如银保监、证监会、公安机关等）
  • 考试时长：行测2小时，申论3小时（专业科目另行通知）

1.3 公告研读要点

  • 仔细阅读职位表：重点关注专业、学历、学位、政治面貌、基层工作经历、服务年限等限制条件
  • 标记意向岗位：将符合条件的岗位名称、代码、招录人数记录在表格中，便于后续对比
  • 注意备注栏：部分岗位有特殊要求（如视力、体能测试、专业能力测试等）

二、网上报名阶段

2.1 报名流程

第一步：注册账号

  1. 登录国家公务员局考录专题网站
  2. 点击"考生报名入口"
  3. 点击"注册新用户"
  4. 填写注册信息：

- 姓名（与身份证一致）

- 身份证号（18位）

- 密码（8-30位，含字母、数字、特殊字符）

- 邮箱（建议使用常用邮箱）

- 验证码

  1. 勾选"我同意"，点击"注册"
  2. 登录邮箱激活账号
注意事项：
  • 注册信息务必准确，后续无法修改姓名和身份证号
  • 密码建议记录在安全位置，避免遗忘

第二步：填写个人信息

登录后点击"完善个人信息"，按顺序填写：

应届生填写要点：
  • 学历：填写当前在读学历（本科/硕士/博士）
  • 毕业时间：填写预计毕业时间（如2026年7月）
  • 专业：按学信网备案名称填写
  • 注意勾选"应届毕业生"选项
在职人员填写要点：
  • 工作经历需详细填写，不得隐瞒
  • 需注明是否同意单位报考意见（部分岗位需单位同意报考证明）
  • 如实填写社保缴纳情况
往届生填写要点：
  • 学历：填写已获得的最高学历
  • 工作经历：如实填写，无工作经历填"待业"
  • 档案存放地：填写档案保管单位名称和地址

第三步：选择岗位

  1. 点击"职位填报"
  2. 输入岗位代码或筛选条件查找意向岗位
  3. 点击"我要报考"
  4. 确认岗位信息：

- 岗位名称、代码、招录人数

- 部门名称、隶属关系

- 工作地点、单位性质

- 专业、学历、政治面貌等要求

  1. 点击"确认报考"
选岗策略：
  • 专业限制优先：优先选择专业要求具体的岗位，减少竞争
  • 基层经历要求：无基层经历者避免选择要求基层经历的岗位
  • 地域因素：考虑工作地点是否接受，是否有户籍限制
  • 招录人数：招录人数多的岗位进面概率相对较高
  • 冷门岗位：条件较为苛刻的岗位竞争相对较小

第四步：等待资格审查

  • 审查时间：提交报名后24-48小时内完成
  • 查询方式：登录报名系统查看"资格审查结果"
  • 状态说明：

- 审核通过：进入下一步

- 审核不通过：根据提示修改信息后重新提交

- 退回补充资料：按要求补充材料后重新提交

2.2 报名注意事项

  1. 只能选择一个岗位：每人限报一个岗位，审核通过后不可修改
  2. 尽早报名：建议在报名截止前3-4天完成报名，预留审核不通过时的修改时间
  3. 信息真实准确：所有信息必须真实，发现虚假信息将取消报考资格
  4. 照片要求：近期免冠蓝底证件照，尺寸符合系统要求
  5. 关注报名人数：部分系统会公布岗位报名人数，可作为参考

三、报名确认及缴费阶段

3.1 确认流程

  1. 登录报名系统
  2. 点击"报名确认及缴费"
  3. 阅读报名确认须知
  4. 上传照片（系统提供照片处理工具）
  5. 选择考点城市（按就近原则）
  6. 确认信息无误后点击"下一步"
  7. 选择缴费方式（支持银联、支付宝等）
  8. 完成缴费

3.2 缴费标准

  • 考试费用：通常每科40-50元，两科共计80-100元（各省标准略有差异）
  • 减免政策：建档立卡贫困家庭人员、城市低保人员可申请减免

3.3 注意事项

  1. 缴费截止时间：逾期未缴费视为自动放弃报考
  2. 考点选择：根据个人情况选择就近考点，一经确认无法修改
  3. 缴费凭证：保留缴费成功截图，以备后续查询

四、打印准考证阶段

4.1 打印流程

  1. 登录报名系统
  2. 点击"打印准考证"
  3. 核对个人信息：姓名、身份证号、准考证号、考试科目、考试时间、考试地点
  4. 选择打印设置：A4纸、黑白/彩色均可
  5. 打印准考证（建议打印2-3份备用）

4.2 准考证使用注意事项

  1. 妥善保管：准考证是进入考场的必备证件，考试期间随身携带
  2. 核对信息：确认考试科目、时间、地点是否正确
  3. 备份电子版：保存准考证电子版，方便随时打印

五、笔试阶段

5.1 考前准备

（1）必备物品

（2）考场规则

  • 提前30分钟进入考场，迟到30分钟不得入场
  • 手机需关机并放入指定位置
  • 答题前填写姓名、准考证号等信息
  • 严格遵守监考人员指令

5.2 考试科目安排（以2026年为例）

5.3 笔试结束后

  • 耐心等待成绩公布（通常在次年1月）
  • 可通过估分工具预估成绩，提前准备面试

六、成绩查询阶段

6.1 查询流程

  1. 登录国家公务员局考录专题网站
  2. 点击"笔试成绩查询"
  3. 输入身份证号和准考证号
  4. 查看成绩：行测分数、申论分数、总分、职位排名（部分省份公布）

6.2 分数线说明

  • 合格分数线：由中央公务员主管部门划定，通常为行测+申论总分不低于90-100分（依年份调整）
  • 进面分数线：各岗位按1:3或1:5比例确定进面人员，分数线因岗位而异

6.3 注意事项

  1. 成绩公布后关注是否进入面试名单
  2. 保存成绩截图，后续资格复审可能需要
  3. 如对成绩有异议，可按规定申请复查

七、资格复审阶段

7.1 复审通知

  • 通知方式：电话、短信、邮件或官网公告
  • 复审材料：根据通知要求准备

7.2 材料清单

（1）通用材料

（2）应届生额外材料

（3）在职人员额外材料

（4）其他特殊材料

7.3 复审流程

  1. 按通知时间到达指定地点
  2. 提交材料，工作人员审核原件并留存复印件
  3. 审核通过后领取面试通知书

7.4 注意事项

  1. 材料务必齐全，缺少材料可能影响资格
  2. 证明文件需加盖公章，复印件需清晰
  3. 按时参加复审，逾期视为自动放弃

八、面试阶段

8.1 面试形式

  • 结构化面试：最常见形式，按固定流程进行
  • 结构化小组面试：部分岗位采用，考生分组答题
  • 无领导小组讨论：部分岗位采用，小组讨论形式

8.2 面试准备

（1）材料准备

  • 面试通知书
  • 身份证
  • 准考证
  • 笔和笔记本

（2）知识准备

  • 复习岗位相关知识
  • 了解报考部门职能
  • 准备自我介绍（3分钟左右）
  • 练习常见面试题型

（3）仪容仪表

  • 着装正式：西装、皮鞋
  • 妆容得体：保持整洁
  • 发型整齐：男生短发，女生束发

8.3 面试流程

  1. 按规定时间到达面试地点
  2. 抽签确定面试顺序
  3. 候考室等待
  4. 进入考场，向考官问好
  5. 听题答题（通常4-5题，20-25分钟）
  6. 答题结束，向考官致谢后离场
  7. 等待成绩公布（部分当场公布）

九、体检阶段

9.1 体检通知

  • 面试结束后，按综合成绩排名确定体检人员
  • 通知方式：电话、短信或官网公告

9.2 体检标准

参照《公务员录用体检通用标准（试行）》和《公务员录用体检操作手册（试行）》

（1）常规项目

（2）特殊岗位要求

9.3 体检注意事项

  1. 体检前准备：

- 体检前3天清淡饮食，避免饮酒

- 体检前1天晚8点后禁食，保持空腹

- 携带身份证和体检通知书

- 女生避开生理期

  1. 体检过程：

- 按引导员指示完成各项检查

- 如实告知病史，不得隐瞒

- 遵守医院规定，保持安静

  1. 体检后：

- 等待体检结果通知

- 如体检不合格，可按规定申请复检

十、政审（考察）阶段

10.1 政审通知

  • 体检合格后进入政审阶段
  • 考察组会联系考生确认相关信息

10.2 政审内容

（1）政治审查

  • 政治立场、政治觉悟
  • 政治历史、有无违法违纪记录
  • 是否参加非法组织

（2）档案审查

  • 学历真实性
  • 工作经历真实性
  • 奖惩情况
  • 个人信息一致性

（3）现实表现考察

  • 应届生：学校表现、学习成绩、奖惩情况
  • 在职人员：工作表现、同事评价、单位意见
  • 社会人员：社区表现、邻里评价

10.3 政审材料清单

（1）应届生材料

（2）在职人员材料

（3）社会人员材料

10.4 政审流程

  1. 考察组与考生联系，说明考察事项
  2. 考生提交相关材料
  3. 考察组查阅档案
  4. 考察组与相关人员谈话了解情况
  5. 考察组出具考察报告

10.5 注意事项

  1. 配合考察组工作，提供真实材料
  2. 保持通讯畅通，及时回复考察组询问
  3. 提前与学校/单位/社区沟通，做好准备

十一、录用公示阶段

11.1 公示内容

  • 拟录用人员名单（姓名、性别、准考证号、报考岗位）
  • 公示期：通常5个工作日
  • 公示平台：国家公务员局官网或招录单位官网

11.2 公示期间

  • 接受社会监督
  • 如有异议，可按规定反映

11.3 公示期满

  • 无异议人员正式录用
  • 发放录用通知

十二、报到入职阶段

12.1 报到准备

  1. 按录用通知要求准备材料：

- 身份证

- 学历学位证书

- 录用通知

- 照片（蓝底免冠）

- 其他要求的材料

  1. 办理档案转移：

- 应届生：学校将档案寄送至录用单位

- 在职人员：从原单位或人才市场调取档案

  1. 办理组织关系转移（党员）：

- 开具组织关系介绍信

- 到新单位党组织报到

12.2 入职手续

  1. 按通知时间到单位报到
  2. 提交报到材料
  3. 签订劳动合同
  4. 办理入职培训手续
  5. 领取工作证件

12.3 试用期

  • 试用期：通常1年
  • 试用期内考核合格后正式任职
  • 考核不合格者取消录用

重要提示

  1. 关注官方信息：所有时间节点和要求以国家公务员局官网发布的公告为准
  2. 保持通讯畅通：报名时填写的手机号在整个招考期间保持畅通
  3. 材料真实完整：所有提交的材料必须真实，发现虚假信息将取消资格
  4. 遵守考试纪律：严格遵守考场规则和招录纪律，违规行为将严肃处理
  5. 及时关注进度：定期登录报名系统查看各阶段状态，避免遗漏重要事项
