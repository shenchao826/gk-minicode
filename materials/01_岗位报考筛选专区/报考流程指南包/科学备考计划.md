## 二、4个月备考计划（全职版）

### 2.1 阶段一：基础攻坚期（第1-6周）

**目标**：搭建知识框架，掌握基本方法

| 周次 | 核心任务 | 每日学习时长 |
|------|----------|--------------|
| 第1-2周 | 资料分析基础+判断推理基础 | 6小时 |
| 第3-4周 | 言语理解基础+数量关系基础 | 6小时 |
| 第5-6周 | 常识判断基础+申论基础 | 6小时 |

**每日时间安排**：

| 时间段 | 学习内容 | 时长 |
|--------|----------|------|
| 7:00-7:30 | 时政晨读（学习强国） | 30分钟 |
| 8:00-10:00 | 行测模块学习（网课+笔记） | 2小时 |
| 10:30-12:00 | 行测模块刷题（对应章节） | 1.5小时 |
| 14:00-16:00 | 申论模块学习（网课+笔记） | 2小时 |
| 16:30-18:00 | 申论练习（1道小题） | 1.5小时 |
| 19:30-21:00 | 当日复盘（笔记整理+错题分析） | 1.5小时 |

**重点**：
- 资料分析优先，投入产出比最高
- 言语理解和判断推理是提分重点
- 申论每天练1道小题，积累规范词

---

### 2.2 阶段二：刷题强化期（第7-10周）

**目标**：提升做题速度和正确率

| 周次 | 核心任务 | 每日学习时长 |
|------|----------|--------------|
| 第7-8周 | 分模块刷真题+错题复盘 | 7小时 |
| 第9-10周 | 混合模块练习+申论大作文 | 7小时 |

**每日时间安排**：

| 时间段 | 学习内容 | 时长 |
|--------|----------|------|
| 7:00-7:30 | 时政晨读+成语背诵 | 30分钟 |
| 8:00-10:30 | 行测真题练习（计时） | 2.5小时 |
| 11:00-12:00 | 行测真题复盘 | 1小时 |
| 14:00-16:00 | 申论练习（1道小题+1篇作文） | 2小时 |
| 16:30-18:00 | 申论复盘+素材积累 | 1.5小时 |
| 19:30-21:30 | 错题重做+薄弱模块补强 | 2小时 |

**重点**：
- 每天刷1套行测真题或2个模块
- 复盘时间≥做题时间
- 申论每周写1篇大作文
- 建立错题本，分类记录

---

### 2.3 阶段三：冲刺模拟期（第11-14周）

**目标**：适应考试节奏，调整答题策略

| 周次 | 核心任务 | 每日学习时长 |
|------|----------|--------------|
| 第11-12周 | 全真模拟+答题顺序调整 | 8小时 |
| 第13-14周 | 高频错题复盘+时政冲刺 | 8小时 |

**每日时间安排**：

| 时间段 | 学习内容 | 时长 |
|--------|----------|------|
| 7:00-7:30 | 时政晨读+公式默写 | 30分钟 |
| 9:00-11:00 | 行测全真模拟（严格计时） | 2小时 |
| 14:00-17:00 | 申论全真模拟（严格计时） | 3小时 |
| 19:00-21:30 | 套卷复盘+错题分析 | 2.5小时 |

**重点**：
- 每周至少2-3次全真模拟
- 模拟后完整复盘，记录每模块用时
- 调整答题顺序，找到最适合自己的策略
- 最后1周减少做题量，回归基础

---

### 2.4 答题顺序建议

| 推荐顺序 | 模块 | 理由 |
|----------|------|------|
| 1 | 资料分析 | 分值高、投入产出比高，放在最前保证正确率 |
| 2 | 判断推理 | 逻辑清晰，容易进入状态 |
| 3 | 言语理解 | 阅读量大，需要保持专注 |
| 4 | 数量关系 | 难度大，放在后面，有时间就做 |
| 5 | 常识判断 | 快速作答，不纠结 |

## 六、时间管理黄金法则

### 6.1 番茄工作法

- 以45-50分钟为一个学习单元
- 中间休息5-10分钟
- 避免连续3小时只学一个模块

### 6.2 "5分钟起步法"

- 不想学习时，告诉自己"只学5分钟"
- 5分钟后，大脑进入状态，自然能继续

### 6.3 每周计划

- 每周日晚上做"下周计划"
- 列出每天要完成的具体任务
- 完成后打勾，获得正向反馈

### 6.4 警惕"假努力"

| 行为 | 本质 | 改进方案 |
|------|------|----------|
| 听课2小时，笔记工整，但从不复习 | 自我感动式学习 | 听完课立刻做练习题，当晚复习笔记 |
| 刷题只做会的，难题跳过 | 舒适区徘徊 | 每套题必须攻克2-3道新题型 |
| 花大量时间找资料、存网盘 | 收集癖 | 只留1-2个来源，其余删除 |
| 每天学10小时，但中途频繁看手机 | 注意力分散 | 学习时手机放另一个房间 |

## 七、各模块提分策略

### 7.1 资料分析（提分最快）

| 策略 | 内容 |
|------|------|
| 优先学 | 投入产出比最高，先攻克 |
| 记公式 | 熟练背诵20+核心公式 |
| 练速算 | 截位直除、百化分每天练 |
| 控时间 | 每篇5-6分钟，4篇25分钟内 |

### 7.2 判断推理（分值最高）

| 策略 | 内容 |
|------|------|
| 图推 | 熟悉特征图，记口诀 |
| 定义 | 抓关键词，对比选项 |
| 类比 | 造句子，找关系 |
| 逻辑 | 记推理规则，不凭感觉 |

### 7.3 言语理解（题量最大）

| 策略 | 内容 |
|------|------|
| 逻辑填空 | 积累高频成语，注意搭配 |
| 片段阅读 | 找主旨句，注意关联词 |
| 细节判断 | 逐项对比，排除干扰 |

### 7.4 数量关系（取舍关键）

| 策略 | 内容 |
|------|------|
| 学简单题型 | 工程问题、行程问题、利润问题 |
| 放弃难题 | 1分钟没思路就跳过 |
| 蒙题技巧 | 选项均匀分布，不会就蒙没选过的 |

### 7.5 常识判断（碎片化积累）

| 策略 | 内容 |
|------|------|
| 日常积累 | 利用碎片时间看时政 |
| 记口诀 | 历史、法律、地理口诀 |
| 不纠结 | 10秒一道题，相信第一感觉 |

### 7.6 申论（抓关键词）

| 策略 | 内容 |
|------|------|
| 小题 | 关键词前置，分层作答 |
| 大作文 | 立意准确，结构清晰 |
| 素材 | 积累规范词和金句 |
| 练字 | 卷面整洁，字体工整 |

## 八、心态管理

### 8.1 设定止损线

- 最多考2-3次，不要无限循环
- 同时准备其他职业方向

### 8.2 保持节奏

- 不要熬夜，保证7小时睡眠
- 每周安排1天休息，不要连续高强度学习

### 8.3 积极心态

- 错题是进步的阶梯，不是失败的证明
- 每次模拟都是发现问题的机会
- 相信自己的努力，坚持就是胜利
