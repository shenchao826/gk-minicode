# 国考岗位智能筛选Excel模板设计说明

## 前言

本说明详细介绍如何制作一个功能强大的国考岗位智能筛选Excel模板，通过数据验证、下拉菜单和IF函数实现多条件一键筛选，帮助考生快速找到符合条件的岗位。

---

## 一、模板整体架构

### 1.1 工作表结构

| 工作表名称 | 用途 |
|-----------|------|
| 原始岗位表 | 存放从官网下载的完整岗位数据 |
| 智能筛选表 | 提供筛选条件输入框和筛选结果展示 |
| 应届生专筛 | 预设应届生条件的筛选结果 |
| 不限专业专筛 | 预设不限专业条件的筛选结果 |
| 基层项目专筛 | 预设基层服务项目定向岗位筛选结果 |

### 1.2 工作流程

```
用户输入筛选条件 → 下拉菜单选择 → IF函数自动匹配 → 筛选结果实时显示 → 用户查看符合条件的岗位
```

### 3.2 设置数据验证

以设置学历下拉菜单为例：

1. 选中智能筛选表中"学历要求"对应的单元格（如B1）
2. 点击"数据"选项卡 → "数据验证" → "数据验证"
3. 在弹出的对话框中：
   - 允许：选择"序列"
   - 来源：输入`=数据源!$A$1:$A$5`
   - 勾选"提供下拉箭头"
   - 点击"确定"

**重复以上步骤**，为其他筛选条件设置下拉菜单：

| 条件单元格 | 数据验证来源 |

| B3（专业） | 直接文本输入，无需下拉 |

## 四、筛选公式编写

### 4.1 核心筛选公式

在智能筛选表的第15行（筛选结果起始行）使用以下数组公式：

#### A15（序号）
```
=IFERROR(ROW()-14,"")
```

#### B15（岗位代码）
```
=IFERROR(INDEX(原始岗位表!$A:$A,SMALL(IF(AND(原始岗位表!$K$2:$K$10000=IF($B$1="不限",原始岗位表!$K$2:$K$10000,$B$1),原始岗位表!$L$2:$L$10000=IF($B$2="不限",原始岗位表!$L$2:$L$10000,$B$2),原始岗位表!$J$2:$J$10000=IF($B$3="",原始岗位表!$J$2:$J$10000,$B$3),原始岗位表!$M$2:$M$10000=IF($B$4="不限",原始岗位表!$M$2:$M$10000,$B$4),原始岗位表!$N$2:$N$10000=IF($B$5="不限",原始岗位表!$N$2:$N$10000,$B$5),原始岗位表!$O$2:$O$10000=IF($B$6="不限",原始岗位表!$O$2:$O$10000,$B$6),原始岗位表!$F$2:$F$10000=IF($B$7="不限",原始岗位表!$F$2:$F$10000,$B$7),原始岗位表!$G$2:$G$10000=IF($B$8="不限",原始岗位表!$G$2:$G$10000,$B$8),原始岗位表!$R$2:$R$10000=IF($B$9="",原始岗位表!$R$2:$R$10000,$B$9)),ROW(原始岗位表!$A$2:$A$10000),999999),ROW()-14)),"")
```

**注意**：输入数组公式后需按 `Ctrl+Shift+Enter` 组合键确认。

### 4.2 简化版筛选公式（使用辅助列）

为了简化公式，建议在原始岗位表中添加辅助列：

#### 辅助列AA（学历匹配）
```
=IF($K2=智能筛选表!$B$1,"匹配",IF(智能筛选表!$B$1="不限","匹配","不匹配"))
```

#### 辅助列AB（学位匹配）
```
=IF($L2=智能筛选表!$B$2,"匹配",IF(智能筛选表!$B$2="不限","匹配","不匹配"))
```

#### 辅助列AC（专业匹配）
```
=IF(智能筛选表!$B$3="","匹配",IF(ISNUMBER(SEARCH(智能筛选表!$B$3,$J2)),"匹配","不匹配"))
```

#### 辅助列AD（政治面貌匹配）
```
=IF($M2=智能筛选表!$B$4,"匹配",IF(智能筛选表!$B$4="不限","匹配","不匹配"))
```

#### 辅助列AE（基层工作年限匹配）
```
=IF($N2=智能筛选表!$B$5,"匹配",IF(智能筛选表!$B$5="不限","匹配","不匹配"))
```

#### 辅助列AF（服务基层项目匹配）
```
=IF($O2=智能筛选表!$B$6,"匹配",IF(智能筛选表!$B$6="不限","匹配","不匹配"))
```

#### 辅助列AG（机构层级匹配）
```
=IF($F2=智能筛选表!$B$7,"匹配",IF(智能筛选表!$B$7="不限","匹配","不匹配"))
```

#### 辅助列AH（职位类别匹配）
```
=IF($G2=智能筛选表!$B$8,"匹配",IF(智能筛选表!$B$8="不限","匹配","不匹配"))
```

#### 辅助列AI（工作地点匹配）
```
=IF(智能筛选表!$B$9="","匹配",IF(ISNUMBER(SEARCH(智能筛选表!$B$9,$R2)),"匹配","不匹配"))
```

#### 辅助列AJ（总匹配结果）
```
=IF(AND(AA2="匹配",AB2="匹配",AC2="匹配",AD2="匹配",AE2="匹配",AF2="匹配",AG2="匹配",AH2="匹配",AI2="匹配"),"符合条件","不符合")
```

### 4.3 使用筛选结果的INDEX公式

在智能筛选表B15单元格输入：

```
=IFERROR(INDEX(原始岗位表!$A$2:$A$10000,SMALL(IF(原始岗位表!$AJ$2:$AJ$10000="符合条件",ROW(原始岗位表!$AJ$2:$AJ$10000)-1),ROW()-14)),"")
```

**按Ctrl+Shift+Enter确认**后，横向拖拽填充其他列，纵向拖拽填充多行。

## 七、使用说明

### 7.1 操作步骤

1. **下载原始岗位表**：从国家公务员局官网下载招考简章Excel文件
2. **复制数据**：将岗位表数据复制到模板的"原始岗位表"工作表中
3. **输入筛选条件**：在"智能筛选表"的条件输入区域选择或输入筛选条件
4. **查看结果**：筛选结果会自动显示在下方的结果区域
5. **切换预设筛选**：点击底部工作表标签切换到应届生专筛、不限专业专筛等

### 7.2 注意事项

1. **数据范围**：公式中的数据范围需根据实际岗位数量调整（当前设置为10000行）
2. **专业匹配**：专业关键词为模糊匹配，输入专业名称的任意部分即可
3. **工作地点匹配**：工作地点为模糊匹配，输入城市名称即可
4. **不限条件**：选择"不限"或留空表示不限制该条件
5. **公式更新**：修改条件后Excel会自动重新计算，确保启用了自动计算功能

### 7.3 常见问题

**Q：筛选结果不显示怎么办？**

A：检查以下几点：
1. 确认原始岗位表中有数据
2. 确认条件输入正确
3. 确认公式已按Ctrl+Shift+Enter确认
4. 确认Excel启用了自动计算（文件→选项→公式→计算选项→自动）

**Q：专业匹配不到怎么办？**

A：尝试以下方法：
1. 输入更简短的关键词（如"计算机"而非"计算机科学与技术"）
2. 确认专业名称拼写正确
3. 检查原始岗位表中的专业字段是否有空格或特殊字符

**Q：如何添加更多筛选条件？**

A：参考现有条件的设置方法：
1. 在数据源工作表中添加新选项

### 8.2 导出筛选结果

添加VBA宏将筛选结果导出为新文件：

```vba
Sub ExportResults()
    Dim ws As Worksheet
    Set ws = ThisWorkbook.Sheets.Add(After:=ThisWorkbook.Sheets(ThisWorkbook.Sheets.Count))
    ws.Name = "筛选结果导出"
    ThisWorkbook.Sheets("智能筛选表").Range("A14:Q10000").Copy
    ws.Range("A1").PasteSpecial Paste:=xlPasteValues
    ws.Range("A1").PasteSpecial Paste:=xlPasteFormats
    Application.CutCopyMode = False
    MsgBox "筛选结果已导出到新工作表！"
End Sub
```

### 8.3 条件保存功能

添加VBA宏保存当前筛选条件：

```vba
Sub SaveFilter()
    Dim filterName As String
    filterName = InputBox("请输入筛选条件名称：")
    If filterName <> "" Then
        ThisWorkbook.Names.Add Name:=filterName, RefersTo:=Range("B1:B10")
        MsgBox "筛选条件已保存为：" & filterName
    End If
End Sub
```
