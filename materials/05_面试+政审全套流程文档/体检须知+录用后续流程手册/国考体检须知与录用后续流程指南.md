# 国考体检须知与录用后续流程指南

## 一、体检概述

### （一）体检目的

- 确保录用人员身体健康，能够胜任工作
- 排除患有重大疾病的人员
- 保障公务员队伍的整体健康水平

### （二）体检依据

- 《公务员录用体检通用标准（试行）》
- 《公务员录用体检操作手册（试行）》

### （三）体检时间

- 一般在面试之后
- 由招录单位统一组织

---

## 二、体检前准备

### （一）饮食准备

| 时间 | 要求 |
|------|------|
| 体检前3天 | 清淡饮食，避免油腻、辛辣食物 |
| 体检前1天 | 晚上8点后禁食，12点后禁水 |
| 体检当天 | 空腹，不进食、不喝水 |

### （二）生活准备

| 项目 | 要求 |
|------|------|
| 休息 | 保证充足睡眠，避免熬夜 |
| 运动 | 避免剧烈运动 |
| 饮酒 | 体检前3天不饮酒 |
| 吸烟 | 体检前1天不吸烟 |

### （三）物品准备

| 物品 | 要求 |
|------|------|
| 身份证 | 原件 |
| 准考证 | 原件 |
| 照片 | 近期免冠照片（1-2张） |
| 签字笔 | 黑色签字笔 |
| 其他 | 根据招录单位要求准备 |

## 三、体检项目

### （一）常规项目

| 项目 | 检查内容 |
|------|----------|
| 身高体重 | 身高、体重测量 |
| 视力 | 裸眼视力、矫正视力 |
| 听力 | 听力测试 |
| 血压 | 血压测量 |
| 心电图 | 心脏功能检查 |
| 胸片 | 肺部检查 |
| 血常规 | 血液指标检查 |
| 尿常规 | 尿液指标检查 |
| 肝功能 | 转氨酶等指标 |
| 肾功能 | 肌酐、尿素氮等指标 |
| 血糖 | 血糖测量 |
| 血脂 | 胆固醇、甘油三酯等指标 |

### （二）特殊项目

| 项目 | 适用人群 |
|------|----------|
| 腹部B超 | 部分岗位 |
| 甲状腺B超 | 部分岗位 |
| 妇科检查 | 女性 |
| 前列腺检查 | 男性 |

## 四、体检常见问题

### （一）常见异常指标

| 指标 | 常见原因 | 应对方法 |
|------|----------|----------|
| 血压偏高 | 紧张、熬夜、饮食不当 | 放松心态、保证睡眠、清淡饮食 |
| 血糖偏高 | 饮食不当、糖尿病 | 控制饮食、适当运动 |
| 转氨酶偏高 | 熬夜、饮酒、脂肪肝 | 保证睡眠、不饮酒、清淡饮食 |
| 视力不达标 | 近视 | 佩戴合适眼镜 |

### （二）不合格情形

| 情形 | 说明 |
|------|------|
| 传染性疾病 | 乙肝、肺结核等 |
| 严重心血管疾病 | 心脏病、高血压等 |
| 严重呼吸系统疾病 | 哮喘、肺气肿等 |
| 严重消化系统疾病 | 胃溃疡、肝炎等 |
| 严重神经系统疾病 | 癫痫、精神病等 |

### （三）复检流程

1. **收到通知**：体检单位通知复检
2. **按时参加**：在规定时间内参加复检
3. **等待结果**：等待复检结果

---

## 五、体检注意事项

### （一）体检中

1. **听从指挥**：按照工作人员指引进行检查
2. **如实告知**：如实告知病史和身体状况
3. **配合检查**：配合医生进行各项检查
4. **保持安静**：保持体检现场安静

### （二）体检后

1. **等待结果**：等待体检结果通知
2. **注意休息**：体检后注意休息，补充营养
3. **保持联系**：保持联系方式畅通

## 七、常见问题解答

### （一）体检相关

**Q1：体检前可以喝水吗？**

A：体检前1天晚上12点后禁水，体检当天早上不喝水。

**Q2：近视会影响体检吗？**

A：一般岗位矫正视力达标即可，特殊岗位要求裸眼视力。

**Q3：体检结果什么时候出来？**

A：一般3-7个工作日，具体以单位通知为准。

### （二）录用相关

**Q1：公示期有人举报怎么办？**

A：招录单位会进行调查，属实则取消录用资格。

**Q2：报到需要准备什么材料？**

A：身份证、毕业证、学位证、照片等，具体以单位通知为准。

**Q3：试用期工资待遇如何？**

A：试用期工资按照国家规定执行，一般为正式工资的80%。
