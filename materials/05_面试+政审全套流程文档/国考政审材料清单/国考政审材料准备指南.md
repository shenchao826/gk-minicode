# 国考政审材料准备指南

## 一、政审概述

### （一）政审目的

- 考察考生的政治素质、道德品质、遵纪守法情况
- 核实考生的学历、工作经历等信息真实性
- 确保录用人员符合公务员录用条件

### （二）政审方式

- **函审**：通过信函调取考生档案和证明材料
- **面审**：工作人员到考生所在单位、学校、社区实地考察

### （三）政审时间

- 一般在面试和体检之后
- 时间跨度约1-2个月

## 五、无犯罪记录证明开具流程

### （一）个人无犯罪记录证明

1. **准备材料**：身份证原件及复印件、户口本
2. **前往派出所**：到户籍所在地派出所申请
3. **填写申请表**：填写《无犯罪记录证明申请表》
4. **审核盖章**：派出所审核后出具证明并盖章

### （二）直系亲属无犯罪记录证明

1. **准备材料**：申请人身份证、亲属关系证明、亲属身份证
2. **前往派出所**：到亲属户籍所在地派出所申请
3. **填写申请表**：填写《无犯罪记录证明申请表》
4. **审核盖章**：派出所审核后出具证明并盖章

## 六、档案调取注意事项

### （一）档案存放地点

| 人群 | 档案存放地点 |
|------|--------------|
| 应届生 | 学校档案馆 |
| 在职人员 | 工作单位或人才中心 |
| 灵活就业人员 | 人才中心或街道办 |

### （二）档案调取流程

1. **开具调档函**：录用单位开具《调档函》
2. **前往存档单位**：携带调档函到档案存放单位
3. **办理调档手续**：存档单位审核后邮寄档案
4. **确认档案到达**：录用单位确认档案收到

### （三）注意事项

1. **档案完整性**：确保档案材料齐全
2. **档案密封**：档案必须密封后邮寄
3. **及时办理**：按照规定时间办理调档手续

## 七、政审常见问题

### （一）常见问题

| 问题 | 说明 |
|------|------|
| 材料不全 | 提前准备，避免遗漏 |
| 档案丢失 | 及时补办，联系相关单位 |
| 无犯罪记录证明难开 | 提前咨询派出所，了解要求 |
| 亲属有犯罪记录 | 根据情况，如实说明 |

### （二）应对策略

1. **提前准备**：了解政审要求，提前准备材料
2. **积极配合**：配合工作人员调查，提供真实信息
3. **及时沟通**：遇到问题及时与录用单位沟通
