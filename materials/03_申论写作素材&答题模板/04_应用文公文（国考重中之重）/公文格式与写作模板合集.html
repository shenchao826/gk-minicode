公文格式与写作模板合集

一、倡议书

（一）格式规范

标题：关于XX的倡议书 / 倡议书 称谓：顶格写，后加冒号（如：广大市民朋友们：） 正文：首行缩进2字符，1.5倍行距 落款：右对齐，倡议单位在上，日期在下

（二）写作结构

  1. 背景引入：阐述问题现状和发起倡议的原因（3-4行）
  2. 倡议内容：分3-4点列出具体行动（每点5-8行）
  3. 呼吁号召：升华主题，发出行动号召（3-4行）
  4. 落款：倡议单位+日期

（三）完整范文

关于开展"文明出行·绿色生活"主题活动的倡议书

广大市民朋友们：

随着城市发展步伐加快，交通拥堵、环境污染等问题日益突出，影响着我们的生活质量和城市形象。为营造文明有序的出行环境，践行绿色低碳的生活方式，我们特此发出如下倡议：

一、文明出行，遵守交通规则。自觉遵守红绿灯信号，不闯红灯、不逆行、不横穿马路；驾驶机动车时礼让斑马线，不随意变道、不车窗抛物；骑行非机动车时佩戴安全头盔，不占用机动车道。让我们从脚下起步，共同维护安全有序的交通秩序。

二、绿色出行，践行低碳生活。优先选择公共交通、步行或骑行等绿色出行方式，减少私家车使用频率；倡导"135"出行模式（1公里步行、3公里骑行、5公里公交），降低碳排放；积极参与"无车日""低碳日"等环保活动，让绿色成为城市最亮丽的底色。

三、文明乘车，共建和谐车厢。排队候车、先下后上，主动为老弱病残孕及抱小孩乘客让座；保持车厢整洁，不随地吐痰、不乱扔垃圾、不大声喧哗；遵守乘车规定，不携带易燃易爆物品，不占座霸座。让每一次出行都成为传递文明的旅程。

四、志愿服务，助力文明创建。积极参与文明交通志愿服务，协助维护交通秩序；主动宣传文明出行理念，带动身边人共同参与；发现不文明行为时善意提醒，共同营造"人人参与、人人共建"的良好氛围。

文明出行一小步，城市进步一大步。让我们携起手来，从自身做起、从现在做起，以实际行动践行文明理念，为建设美丽宜居的现代化城市贡献力量！

倡议单位：XX市文明办

XX市交通运输局

XXXX年XX月XX日

（四）真题示例

真题：给定资料中提到"白色污染"问题日益严重，请你以环保部门名义，写一份向市民发出的环保倡议。（40分） 高分作答思路：
  1. 开头点明白色污染现状（数据支撑：每年使用塑料袋XX个，降解时间XX年）
  2. 分点列出具体措施（减少使用、分类回收、替代方案、宣传教育）
  3. 结合"绿水青山就是金山银山"等政策表述
  4. 结尾呼吁全民参与，共建美丽家园
常见失分点：
  • 只提口号，没有具体行动措施
  • 语言口语化，缺乏正式感
  • 倡议内容空洞，没有可操作性

二、公开信

（一）格式规范

标题：致XX的一封公开信 / 公开信 称谓：顶格写，后加冒号（如：尊敬的市民朋友：） 正文：首行缩进2字符，1.5倍行距 结尾：此致，敬礼！（换行右对齐） 落款：署名+日期

（二）写作结构

  1. 问候与身份介绍（2-3行）
  2. 正文内容：陈述事实、表达观点或提出建议（重点部分）
  3. 结尾：表达期望或感谢（2-3行）
  4. 落款：署名+日期

（四）真题示例

真题：假设你是某市政府工作人员，针对市民反映的"停车难"问题，写一封公开信回应市民关切。（40分） 高分作答思路：
  1. 先表达理解和歉意，拉近与市民距离
  2. 说明政府已采取的措施（新增车位、智慧停车、错峰停车）
  3. 承认存在的问题，说明正在推进的工作
  4. 呼吁市民理性停车、文明停车
  5. 提供反馈渠道，体现服务意识
常见失分点：
  • 语气生硬，缺乏沟通感
  • 回避问题，只讲成绩
  • 没有具体措施，空话套话

  4. 下一步计划：提出具体的改进措施和工作安排（3-4点）
  5. 结尾：请求领导批示

关于我市社区养老服务体系建设情况的调研报告

为深入了解我市社区养老服务体系建设现状，分析存在问题，提出改进建议，我们组织开展了专项调研。调研采取实地走访、问卷调查、座谈交流等方式，覆盖XX个社区、XX家养老服务机构，发放问卷XX份，现将调研情况报告如下：

一、现状分析

（一）政策体系逐步完善。我市先后出台《社区养老服务体系建设规划》《居家养老服务条例》等政策文件，明确了社区养老服务的发展目标和保障措施。建立了政府主导、社会参与、市场运作的养老服务机制。

（二）服务网络初步形成。全市已建成社区养老服务中心XX个、日间照料中心XX个、居家养老服务站点XX个，基本实现城市社区全覆盖。服务内容涵盖生活照料、医疗护理、精神慰藉、文化娱乐等多个方面。

（三）服务质量稳步提升。引入专业养老服务机构参与运营，开展养老服务人员培训，服务标准化、规范化水平不断提高。部分社区探索"互联网+养老"模式，通过智能平台为老年人提供便捷服务。

（四）社会参与度不断提高。鼓励企业、社会组织参与社区养老服务，形成了多元化的服务供给格局。志愿者队伍不断壮大，为老年人提供志愿服务XX万人次。

二、存在问题

（一）供需矛盾突出。随着老龄化程度加深，养老服务需求持续增长，但服务供给总量不足、结构不合理，尤其是针对高龄、失能、独居老人的专业护理服务短缺。

（二）资源整合不够。养老服务资源分散在民政、卫健、医保等多个部门，缺乏有效整合，存在重复建设和资源浪费现象。

（三）专业人才匮乏。养老服务人员待遇偏低、职业发展空间有限，导致人才流失严重，专业护理人员缺口较大。

（四）资金投入不足。财政投入难以满足养老服务发展需求，社会资本参与养老服务的积极性不高，融资渠道有限。

三、对策建议

（一）完善服务体系，扩大供给规模。编制养老服务设施专项规划，合理布局养老服务设施；支持社会力量兴办养老服务机构，鼓励家政服务企业拓展养老服务业务；加强对养老服务质量的监管，确保服务规范。

（二）推进资源整合，提高服务效能。建立跨部门协调机制，整合民政、卫健、医保等部门资源，实现信息共享、服务协同；推进医养结合，加强养老机构与医疗机构的合作，为老年人提供一体化的健康养老服务。

（三）加强人才培养，提升服务能力。建立养老服务人才培训体系，开展职业技能培训和等级认定；提高养老服务人员待遇，完善激励机制；吸引专业人才投身养老服务事业，打造高素质的养老服务队伍。

（四）拓宽资金渠道，强化保障措施。加大财政投入力度，设立养老服务专项基金；鼓励金融机构创新养老服务金融产品，支持养老服务产业发展；引导社会资本参与养老服务设施建设和运营。

社区养老服务体系建设是保障民生、促进社会和谐的重要举措。我们将以此次调研为契机，认真落实各项措施，不断完善社区养老服务体系，为老年人提供更加优质、高效、便捷的养老服务。

关于开展安全生产大检查的通知

各区县人民政府、各有关部门：

为深入贯彻落实习近平总书记关于安全生产工作的重要指示精神，深刻汲取近期安全事故教训，切实保障人民群众生命财产安全，经市政府研究决定，在全市范围内开展安全生产大检查。现将有关事项通知如下：

一、检查范围

全市所有生产经营单位，重点检查危险化学品、烟花爆竹、矿山、建筑施工、交通运输、消防安全、特种设备等行业领域。

二、检查内容

（一）安全生产责任制落实情况。检查企业是否建立健全安全生产责任制，是否明确各岗位安全职责，是否落实"一岗双责"。

（二）安全管理制度执行情况。检查企业是否制定完善安全管理制度和操作规程，是否严格执行安全教育培训、隐患排查治理、应急管理等制度。

（三）安全隐患排查整改情况。检查企业是否开展经常性安全隐患排查，对发现的隐患是否及时整改到位，是否建立隐患排查治理台账。

（四）安全设施设备运行情况。检查企业安全设施设备是否完好有效，是否定期进行维护保养和检测检验，是否存在带病运行情况。

真题：某社区招募疫情防控志愿者，撰写招募公告。（30分） 高分作答思路：
  1. 招募背景要紧迫（疫情防控形势严峻，需要志愿者支持）
  2. 招募条件要明确（年龄、健康状况、服务能力）
  3. 服务内容要具体（核酸检测、物资配送、政策宣传）

  5. 报名方式要便捷（线上线下结合）
常见失分点：
  • 招募条件过于苛刻或过于宽松
  • 服务内容描述模糊
  • 保障措施不明确，影响招募效果

（一）格式规范

十六、公文写作练习题库

（一）练习题1

题目：假设你是某市文明办工作人员，针对城市不文明养犬问题，写一份倡议书。 要求：
  1. 格式规范，语言得体
  2. 内容具体，措施可行
  3. 字数不少于500字
参考答案要点：
  1. 背景：不文明养犬现象（不牵绳、不清理粪便、犬只扰民）
  2. 倡议内容：牵绳遛犬、清理粪便、定期防疫、文明训导、遵守规定
  3. 呼吁：共同营造文明养犬氛围

（二）练习题2

题目：某市政府决定开展安全生产大检查，撰写通知。 要求：
  1. 格式规范，要素齐全
  2. 内容具体，要求明确
  3. 字数不少于500字
参考答案要点：
  1. 发文缘由：近期安全事故频发
  2. 检查范围：所有行业领域
  3. 检查内容：责任制落实、隐患排查、设施设备、应急管理
  4. 时间安排：自查→检查→督查→总结
  5. 工作要求：加强领导、严格执法、及时上报

（三）练习题3

题目：调研某县农村电子商务发展情况，撰写调研报告。 要求：
  1. 结构完整，逻辑清晰
  2. 内容详实，数据支撑
  3. 对策可行，针对性强
  4. 字数不少于800字
参考答案要点：
  1. 现状分析：发展规模、特色产业、基础设施
  2. 存在问题：人才短缺、物流不畅、品牌不强、标准不一
  3. 对策建议：加强人才培养、完善物流体系、打造特色品牌、制定行业标准

（四）练习题4

题目：作为某县教育局局长，在全县教育工作会议上发表讲话。 要求：
  1. 结构清晰，层次分明
  2. 内容充实，有数据支撑
  3. 语言得体，有感染力
  4. 字数不少于800字
参考答案要点：
  1. 总结成绩：教育投入、师资建设、教育质量、均衡发展
  2. 分析问题：城乡差距、师资短缺、教育质量有待提升
  3. 部署工作：立德树人、优质均衡、教师队伍建设、教育改革
  4. 强调要求：加强领导、压实责任、强化督导
